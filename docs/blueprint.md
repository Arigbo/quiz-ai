# **App Name**: QuizSolver AI Plugin

## Core Features:

- Automatic Question Detection: Automatically scans and identifies quiz questions and multiple-choice options directly within the active browser window or document viewer.
- In-Situ AI Reasoning Tool: Utilizes a generative AI model as a tool to reason through the detected context of a question and determine the most accurate answer without requiring manual input.
- Direct Answer Overlay: Overlays the AI-generated answer or the correct choice directly next to the question element for instant verification during testing.
- Auto-Select Functionality: Automatically clicks the radio button or checkbox corresponding to the AI's identified correct answer to streamline the testing flow.

## Style Guidelines:

- Professional and focused palette: Primary Insightful Blue (#266DD2) for UI elements, Serene Off-White (#F0F4F7) for backgrounds, and Active Cyan (#33B2E0) for answer highlighting.
- The 'Inter' sans-serif font is used throughout to provide a modern, objective, and highly legible experience for both questions and generated answers.
- Minimalist, line-based icons for plugin controls such as 'Enable/Disable' and 'Refresh Scan' to minimize visual clutter on the page.
- A floating sidebar and context-aware tooltips that appear near detected questions, ensuring the plugin remains unobtrusive while providing necessary data.
- Subtle fade-in animations for answer overlays and gentle pulsing effects on detected elements to indicate processing activity.
